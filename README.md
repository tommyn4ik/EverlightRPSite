# DesertRP - Minecraft RolePlay Website

Добро пожаловать в DesertRP - уникальный Minecraft RolePlay сервер в стиле Дикого Запада. Этот веб-сайт представляет проект и предоставляет всю необходимую информацию для игроков.

## Особенности сайта

- **Современный дизайн** - стильный и привлекательный интерфейс в стиле темного Дикого Запада
- **Адаптивная верстка** - корректное отображение на всех устройствах (ПК, планшеты, смартфоны)
- **Интерактивные элементы** - анимации, плавные переходы, интерактивные карточки классов
- **Полноценная навигация** - удобное меню с плавающим скроллом
- **Информативные разделы** - о проекте, игровые классы, правила, информация о сервере
- **Форма обратной связи** - возможность связи с администрацией

## Структура проекта

```
/workspace/
├── index.html          # Главная страница сайта
├── styles.css          # Стили оформления
├── script.js           # Скрипты для интерактивности
└── README.md           # Документация (этот файл)
```

## Основные разделы сайта

1. **Главная страница** - Приветствие и призыв к действию
2. **О проекте** - Информация о сервере и его особенностях
3. **Игровые классы** - Подробное описание всех доступных ролей
4. **Правила** - Основные правила поведения на сервере
5. **Сервер** - Техническая информация и статус онлайн
6. **Контакты** - Способы связи с администрацией

## Используемые технологии

- HTML5
- CSS3 (с адаптивной версткой и анимациями)
- JavaScript (для интерактивности)
- Font Awesome (иконки)
- Google Fonts (шрифты)

## Установка и запуск

1. Скопируйте все файлы в папку вашего веб-сервера
2. Убедитесь, что все файлы находятся в одной директории
3. Откройте `index.html` в веб-браузере

## Особенности дизайна

- Цветовая схема в стиле темного Дикого Запада (#2c3e50, #3498db, #e74c3c)
- Использование градиентов и теней для глубины
- Анимации при наведении и скролле
- Адаптивное меню для мобильных устройств

- Добавление галереи скриншотов
